# 🌱 Irrigação Inteligente e Sustentável - RecySprout
3º Lugar do Ideathon UNIVAG | Programa Inova Cerrado MT 2024

Este repositório contém o código-fonte da landing page da RecySprout. O projeto nasceu como uma proposta de startup para levar tecnologia de baixo custo ao pequeno agricultor, unindo automação e sustentabilidade.


## 🖥️ Sobre a Interface

### Seções Principais
- Hero/Banner: Apresentação de impacto com foco no propósito de revolucionar a agricultura.
- Solução: Grid explicativo sobre o funcionamento do irrigador e economia de recursos.
- Data & Analytics: Seção visual que utiliza dados para contextualizar a necessidade do produto.
- Showcase do Protótipo: Demonstração das funcionalidades técnicas.
- Modelo de Negócios: Tabela de planos com diferenciais de serviço.
- Equipe: Espaço dedicado aos desenvolvedores e idealizadores do projeto.
